# (PRE-RELEASE) Go tools for [Sourcegraph](https://sourcegraph.com) [![Build Status](https://travis-ci.org/sourcegraph/sourcegraph-go.svg)](https://travis-ci.org/sourcegraph/sourcegraph-go)
